# Copilot Instructions for CV Parser

## Project Overview
This is a web application that reads a new employee's CV and IFU, extracts skills, and saves them in a structured format.

## Architecture
- **Backend**: .NET 10 Web API (located in `src/CvParser.Api`)
- **Frontend**: React with Vite (located in `src/CvParser.Web`)

## Development Environment
The project supports the following IDEs:
- Visual Studio
- JetBrains Rider
- Visual Studio Code

## Getting Started

### Backend (.NET Web API)
```bash
cd src/CvParser.Api
dotnet restore
dotnet build
dotnet run
```

### Frontend (React)
```bash
cd src/CvParser.Web
npm install
npm run dev
```
